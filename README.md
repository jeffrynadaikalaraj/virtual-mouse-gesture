# ProjectHub - Project Upload & Showcase Website

A beautiful, modern website for uploading, showcasing, and sharing your coding projects with special support for Jupyter notebooks and various file types.

## 🚀 Features

### 🎯 Core Functionality
- **Drag & Drop Upload**: Simply drag your files onto the upload area or click to browse
- **Multi-file Support**: Upload multiple files at once for complete projects
- **Project Gallery**: Browse all uploaded projects with filtering capabilities
- **Real-time Preview**: View Jupyter notebooks, code files, and other content directly in the browser
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

### 📁 Supported File Types
- **Jupyter Notebooks** (`.ipynb`) - Full notebook rendering with code and markdown cells
- **Python Files** (`.py`) - Syntax-highlighted code preview
- **JavaScript** (`.js`) - Formatted code display
- **Web Files** (`.html`, `.css`) - Styled code preview
- **Markdown** (`.md`) - Rich text formatting
- **Archives** (`.zip`) - File information display
- **JSON** (`.json`) - Formatted data display

### 🎨 Modern UI Features
- **Beautiful Animations**: Smooth transitions and hover effects
- **Glass Morphism**: Modern backdrop blur effects
- **Gradient Backgrounds**: Eye-catching color schemes
- **Interactive Elements**: Floating cards and animated components
- **Dark Code Themes**: Professional code display

## 🛠️ How to Use

### 1. Opening the Website
Simply open `index.html` in your web browser. The website works entirely in the browser with no server required!

### 2. Uploading Projects
1. **Navigate to Upload Section**: Click "Upload Project" or scroll to the upload section
2. **Add Files**: 
   - Drag and drop files onto the upload area, OR
   - Click the upload area to browse and select files
3. **Fill Project Details**:
   - **Title**: Give your project a descriptive name
   - **Description**: Explain what your project does
   - **Tags**: Add relevant tags (e.g., "python, machine-learning, data-science")
4. **Upload**: Click the "Upload Project" button
5. **Progress**: Watch the upload progress bar complete
6. **Success**: Your project is now in the gallery!

### 3. Browsing Projects
1. **Gallery Section**: Scroll to the gallery or click "View Gallery"
2. **Filter Projects**: Use the filter buttons to show specific types:
   - All
   - Python
   - JavaScript
   - Data Science
   - Web Dev
3. **View Details**: Click any project card to open detailed view

### 4. Project Details Modal
When you click a project, you'll see:
- **Project Information**: Title, description, tags, upload date
- **File Preview**: 
  - Jupyter notebooks with formatted cells
  - Code files with syntax highlighting
  - File information for other types
- **Actions**: Download and share buttons

## 💾 Data Storage

Projects are stored locally in your browser using localStorage, which means:
- ✅ **No server required** - works offline
- ✅ **Fast access** - instant loading
- ✅ **Privacy** - your data stays on your device
- ⚠️ **Local only** - projects won't sync across devices
